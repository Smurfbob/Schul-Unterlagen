# IPv6

+ Besteht aus acht Blöcken

+ Jeder Block besteht aus zwei Bytes

+ Zwei hoch 128 Adressen sind möglich

+ Die IPv6 wird in Hexadezimal geschrieben

```
Beispiel IPv6: 0000.0000.0000.2135.A201.00FD.DCEF.125A
```
